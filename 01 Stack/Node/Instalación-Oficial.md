A continuación se muestran los pasos a seguir para la instalación de node.js en un entorno nuevo
Fuente: [YouTube](https://www.youtube.com/watch?v=yB4n_K7dZV8&list=PLUofhDIg_38qm2oPOV-IRTTEKyrVBBaU7)
### Pasos para la instalación de `node.js` de la manera oficial
1. Instalar Node.js desde el sitio oficial (Versión LTS)
	[Descargar](https://nodejs.org/en/download)
#### Nota:
Esta opción instala una versión especifica de node en el sistema operativo. Lo cual no es conveniente cuando se trabaja en multiples proyectos.
### Ejecución 
Para ejecutar un archivo de node se utiliza el comando `node`
```PowerShell
	node archivo.js
```
