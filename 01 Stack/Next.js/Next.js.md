# Next.js

Framework React para producción con renderizado híbrido (SSR, SSG, CSR).

- **Docs**: [nextjs.org/docs](https://nextjs.org/docs)
- **Deploy**: Vercel

## Características

- App Router (recommended)
- Server Components
- API Routes
- TypeScript integrado
