# IA Tools

Herramientas de inteligencia artificial para desarrollo de software. Incluye agentes, asistentes y extensiones.

---

## 🛠️ Herramientas Disponibles

### OpenCode
[[OpenCode.md]] - Agente de terminal con IA (Recomendado)
- CLI interactivo con IA
- Exploración de codebase
- Ejecución de comandos
- Automatización de tareas
- Integración con git

### Claude Code
[[Claude-Code.md]] - Extensión de IDE para Claude
- Análisis de código
- Refactoring automático
- Generación de código
- Explicación de lógica
- Implementación de features

### GitHub Copilot
[[GitHub-Copilot.md]] - Asistente de código de GitHub
- Autocompletado inteligente
- Sugerencias de código
- Generación de tests
- Documentación automática
- Soporte multi-lenguaje

---

## 📊 Comparativa

| Feature | OpenCode | Claude Code | GitHub Copilot |
|---------|----------|-------------|-----------------|
| **Tipo** | Agent terminal | IDE extension | IDE extension |
| **Modelo** | Claude | Claude | GPT-4 / Codex |
| **Interfaz** | CLI | GUI | Inline |
| **Codebase** | Exploración profunda | Local | Contextual |
| **Automatización** | Alta | Media | Media |
| **Mejor para** | DevOps, investigación | Análisis, refactoring | Autocompletado |

---

## 🚀 Casos de Uso

### OpenCode
- Exploración de proyectos desconocidos
- Automatización de tareas repetitivas
- Análisis de bugs complejos
- Implementación de features
- Refactoring masivo

### Claude Code
- Code review asistido
- Refactoring inteligente
- Generación de documentación
- Implementación de funcionalidades
- Explicación de código legado

### GitHub Copilot
- Autocompletado de código
- Generación de tests
- Snippets de código
- Completado de documentación
- Aprendizaje de APIs
